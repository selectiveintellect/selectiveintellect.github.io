The Elevator Pitch

Parallel Solution Engine (or PARASOL™) is an operating system agnostic middleware framework to enable users to easily leverage heterogeneous systems with GPUs and CPUs to achieve efficient parallelization of their algorithms.

Why Should You Use PARASOL™ ?

For HPC Developers


PARASOL™ will

  • Abstract communications and events between compute devices
  • Allow mixing of various compute devices
  • Provide an operating system agnostic cross platform API
  • Allow dynamic creation of a cluster
  • Perform intelligent workload distribution

... and ...

  • Simplify implementation complexity
  • Reduce Time to Market for your software
  • Increase time available to on your algorithms

PARASOL™ Implements

  • Communications between heterogeneous systems
  • Fault tolerance
  • Intelligent Workload balancing

Problems Solved With PARASOL™

  • Massively parallel
  • Embarassingly parallel
  • Graph parallel
  • Custom domain decomposition algorithms

If You Are a System Administrator...

then PARASOL™ does the following for you

  • Maximizes the available resources provided to it
  • Performs workload balancing and fault tolerance
  • Works with existing HPC clusters that use MPI
  • Allows you to mix old servers with newer ones to create a cluster
  • Integrates with existing monitoring systems such as Nagios, Cacti and Ganglia, by using optional SNMP messaging

Supports Latest HPC Technology

PARASOL™ supports the latest advances in HPC technology:

  • Graphical Processing Units(GPUs)
  • Multi-processor and Multi-core CPUs
  • Field Programmable Gate Array (FPGA) cards

Intelligent Workload Balancing


PARASOL™ solves this common problem by:

  • Retrieving information on each compute device
  • Measuring the performance of the compute device
  • Allocating more work to compute devices which perform better
  • Handling failure of compute devices and recovers seamlessly
  • Automatic adjusting of workload based on performance

Supported Hardware


Currently systems capable of running OpenCL 1.2 are supported:


  • NVIDIA® GPUs
  • AMD® GPUs and CPUs
  • Intel® CPUs

Supported Operating Systems


PARASOL™ is operating system agnostic and supports a cluster of heterogeneous systems running the following:


  • 64-bit Linux®
  • 64-bit Microsoft Windows®

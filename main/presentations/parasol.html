PARASOL™

A Parallel Solution Engine for HPC

Created by Selective Intellect / @selectintellect

The Elevator Pitch

Parallel Solution Engine (or PARASOL™) is an operating system agnostic middleware framework to enable users to easily leverage heterogeneous systems with GPUs and CPUs to achieve efficient parallelization of their algorithms.

Why Should You Use PARASOL™ ?

If You Are an HPC Developer...

... then PARASOL™ does the following for you

  • Abstracts out low level data exchange and event handling across compute devices
  • Provides a stable Cross Platform API to be operating system agnostic
  • Allows mixing of compute devices and operating systems to create a cluster dynamically
  • Performs intelligent workload distribution between compute devices

... and ...

  • Simplifies implementation complexity
  • Gives you time to focus on your algorithm instead of:
    • implementing communications between systems
    • implementing fault tolerance
    • implementing workload balancing
  • Reduces Time to Market for your software
  • Supports the following types of problems:
    • massively parallel
    • embarassingly parallel
    • graph parallel
    • custom domain decomposition algorithms

If You Are a System Administrator...

then PARASOL™ does the following for you

  • Maximizes the available resources provided to it
  • Performs workload balancing and fault tolerance
  • Works with existing HPC clusters that use MPI
  • Allows you to mix old servers with newer ones to create a cluster
  • Integrates with existing monitoring systems such as Nagios, Cacti and Ganglia, by using optional SNMP messaging

Supports Latest HPC Technology

PARASOL™ supports the latest advances in HPC technology:

  • Graphical Processing Units(GPUs)
  • Multi-processor and Multi-core CPUs
  • Field Programmable Gate Array (FPGA) cards

Intelligent Workload Balancing

Workload Balancing is a common problem and PARASOL™ handles it well:

  • It retrieves information on each compute device
  • It measures the performance of the compute device
  • It allocates more work to compute devices which perform better
  • It handles failure of compute devices and recovers seamlessly
  • It can detect change in loads of a compute system and adjust the workload

Supported Hardware


Currently systems capable of running OpenCL 1.2 are supported:


  • NVIDIA® GPUs
  • AMD® GPUs and CPUs
  • Intel® CPUs

Supported Operating Systems


PARASOL™ is operating system agnostic and supports a cluster of heterogeneous systems running the following:


  • 64-bit Linux®
  • 64-bit Microsoft Windows®
